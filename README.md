# 🎵 Paniccroomstudio

Web oficial de **Paniccroomstudio**, un estudio musical. La aplicación presenta el estudio y permite a los usuarios ponerse en contacto con el propietario.

## 🚀 Tecnologías

- **Backend:** Spring Boot (Java)
- **Frontend:** Thymeleaf, JavaScript

## ✨ Funcionalidades

- Presentación del estudio musical
- Formulario de contacto con el dueño

## ▶️ Cómo ejecutar el proyecto

### Requisitos previos

- Java 17+
- Maven

### Pasos

```bash
# Clonar el repositorio
git clone https://github.com/denmicc73/Paniccroomstudio.git

# Entrar en el directorio
cd Paniccroomstudio

# Ejecutar la aplicación
./mvnw spring-boot:run
```

La aplicación estará disponible en `http://localhost:8080`
